Install
------------------------------------------------------------------------------

``learn_hubspot`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install learn-hubspot

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade learn-hubspot
